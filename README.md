===============================
 🔥 WebP to JPG 변환기 🔥
===============================

📁 파일 설명:
- 설치.bat              : 라이브러리 설치
- 실행.bat              : 해커 모드 실행 (스마트 자동 실행)

💡 사용 팁:

✅ 드래그 앤 드롭이 안 되면?
   → "파일 선택" 버튼 사용

✅ 여러 파일 처리?
   → Ctrl+클릭으로 다중 선택

✅ 한글이 깨지면?
   → 설치.bat / 실행.bat 사용

✅ 오류가 나면?
   → Python 설치 확인
   → 설치.bat 다시 실행

📞 문제 해결:
- 드래그앤드롭_문제해결.md 파일 참고
- Python이 설치되어 있어야 함
- 관리자 권한으로 실행 시도
